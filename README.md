# tensorboard-embedding-visualization

Visualize embeddings in Tensorboard

Generate embeddings:

	python embedding.py --output-dir /tmp/my_embedding

Start Tensorboard:

	python embedding.py --output-dir /tmp/my_embedding

Open a Browser at http://127.0.0.1:6006 and go to the Projector tab
